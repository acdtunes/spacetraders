## spacetraders captain wake watch clear

Disarm all one-shot wake watches

### Synopsis

Disarm every currently-armed one-shot wake watch. The standing wake policy
("captain wake set") is unaffected.

Examples:
  spacetraders captain wake watch clear

```
spacetraders captain wake watch clear [flags]
```

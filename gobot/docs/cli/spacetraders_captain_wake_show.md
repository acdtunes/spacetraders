## spacetraders captain wake show

Show the captain's currently declared wake policy

### Synopsis

Show the wake policy currently declared via "captain wake set", or the
defaults if none has been declared.

Examples:
  spacetraders captain wake show

```
spacetraders captain wake show [flags]
```

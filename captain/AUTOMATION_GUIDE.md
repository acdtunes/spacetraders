# Retired — automation proposals moved to beads

This guide described the pre-city-bridge proposal flow (`reports/bugs/*.md`
frontmatter). That engine is retired.

Automations are proposed as beads now: `bd create -t feature -l shipwright` with
acceptance criteria (big features additionally need the Admiral's `bd human`
approval before code — RULINGS #9/Tier-3). The container engine itself is
documented in `CLI-PRIMER.md` (repo root) and the code.

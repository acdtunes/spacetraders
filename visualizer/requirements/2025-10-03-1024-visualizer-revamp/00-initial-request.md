## Context

This request is for enhancing the SpaceTraders Fleet Visualization application - a real-time web app that tracks agent ship movements across star systems in the SpaceTraders game.

### Current Features
- Real-time ship tracking (5-second polling)
- Interactive 2D space map with PixiJS
- Multi-agent support with color-coded ships
- Ship details (fuel, cargo, navigation status)
- Basic filtering by ship status or agent
- System switching between star systems

### Technology Stack
- Frontend: React 18, TypeScript, PixiJS, Zustand, Tailwind CSS, Vite
- Backend: Express.js, TypeScript, JSON file storage

## Goal

Identify and implement interesting features that will significantly improve the visualizer's usefulness, user experience, and visual appeal for SpaceTraders players managing their fleets.
